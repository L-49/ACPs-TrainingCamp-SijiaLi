# ACPs-TrainingCamp-李思佳

这是一个用于ACPs实训营申请的代码仓库，包含个人编写的智能体相关代码项目。

## 项目介绍

### DSL Agent项目
基于领域特定语言(DSL)的智能客服Agent，通过大语言模型(LLM)意图识别驱动DSL脚本执行，支持电商和技术支持两大业务场景。

## 核心功能

- **多业务场景支持**：电商客服 + 技术支持
- **意图识别**：基于DashScope Qwen API的智能意图识别
- **DSL规则引擎**：基于YAML的领域特定语言，支持动态规则配置
- **动态回复生成**：LLM生成个性化回复
- **对话历史管理**：完整的对话历史记录和管理

## 技术实现

- **语言**：Python
- **DSL格式**：YAML
- **意图识别**：DashScope Qwen API
- **项目结构**：模块化设计，清晰的代码组织
